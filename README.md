# Microsoft UWP fonts

**Link: <https://bit.ly/UWP-fonts>**

## Download fonts

Go to the [`fonts`](./fonts/) directory to download the fonts.

* Sans-serif fonts
	* Arial

	* Consolas
	* Segoe UI

* Serif fonts
	* Cambria
	* Courier New
	* Georgia
	* Times New Roman
* Variable fonts
	* Bahnschrift
	* Segoe UI Variable
* Symbols and icons
	* Segoe MDL2 Assets
	* Segoe UI Emoji
	* Segoe UI Symbol
